# DTU Dataset Manager

## Download

download the DTU dataset

```bash
https://roboimagedata.compute.dtu.dk/?page_id=36
```

quick start

```bash
http://roboimagedata2.compute.dtu.dk/data/MVS/Rectified.zip
-> <path-to-DTU-dataset-folder>/Rectified/
```

## Setup

```bash
conda create -n dtu python=3.10
conda activate dtu
./setup.sh
```

## Run

```bash
python demo.py
```

## Enjoy it~
